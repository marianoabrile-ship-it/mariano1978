# 🏢 Agente Alquileres Córdoba

Agregador de alquileres de Córdoba, Argentina. Centraliza Zonaprop, Argenprop, MercadoLibre e inmobiliarias locales.

## Deploy en Railway

1. Subí este repo a GitHub
2. Entrá a [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Seleccioná el repo → Railway detecta todo automáticamente
4. En 3 minutos tenés la URL pública

## Correr localmente

```bash
cd backend
pip install -r requirements.txt
playwright install chromium
python3 api.py
```

Dashboard en http://localhost:8080
